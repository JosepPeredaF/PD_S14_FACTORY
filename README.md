# Factory Method
### implementacion de patron de diseño 

* Contexto: 

  La compañía de teléfonos Línea Rápida ofrece tres planes de datos: Personal, Negocio y Académico. Cada plan tiene una tasa de pago diferente por mega consumida: 0.5, 1.5 y 1.0 sol respectivamente.
  Además, se incluyen dos proveedores adicionales: Claro, con tasas de 0.6, 1.6 y 1.2 sol por mega, y Movistar, con tasas de 0.8, 1.6 y 1.25 sol por mega.
